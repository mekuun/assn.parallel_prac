# Практикум по параллельному программированию

В репозитории собраны учебные задачи по параллельному программированию на C и C++.

Основные темы:

- POSIX Threads;
- OpenMP;
- MPI;
- векторизация с ARM NEON;
- измерение производительности и кеш-промахов через PAPI;
- параллельные алгоритмы для массивов, матриц, графов и сеточных задач.

## Содержание

| Файл | Тема |
| --- | --- |
| `task1.c` | Параллельное вычисление числа pi с использованием POSIX Threads |
| `task2.cpp` | Потокобезопасная очередь на mutex и condition variable |
| `task3.cpp` | Обработка графа в CSR-формате и измерение кеш-промахов через PAPI |
| `task4.cpp` | Умножение матриц: последовательная версия и SIMD-версия на ARM NEON |
| `task5.cpp` | Моделирование случайного блуждания с OpenMP |
| `task6.cpp` | Параллельная сортировка слиянием с OpenMP tasks |
| `task7.cpp` | MPI-решение сеточной задачи методом итераций |
| `task8.cpp` | MPI-реализация игры Conway's Game of Life |
| `task9.cpp` | MPI-умножение матриц с блочным разбиением процессов |
| `task10.cpp` | 3D Jacobi-итерации с MPI Cartesian topology и производными типами данных |
| `task11.cpp` | MPI one-sided communication для умножения матрицы на вектор |

PDF-файлы рядом с исходниками содержат условия соответствующих задач.

## Технологии

- C / C++;
- pthreads;
- OpenMP;
- MPI;
- ARM NEON intrinsics;
- PAPI.

## Структура

```text
.
├── task1.c
├── task2.cpp
├── ...
├── task11.cpp
└── task*.pdf
```

Каждый `task*.c` или `task*.cpp` является отдельной учебной задачей.
